# minecraft-avenoobs
Our minecraft homelab


## Dependecies
- docker-compose
- npm
- vtop
- tmux

#### vtop install

> npm install -g vtop


#### Run it

> docker-compose up -d

#### Monitor
> bin/minecraft-tmux
